# Clean up the resources in the Cloud

* Run the code below to destroy the whole AKS cluster

    ```sh
    docker container run -it --rm \
        --env-file $HOME/jumphost/workspace/.tf_creds \
        -v $HOME/jumphost/workspace:/workspace \
        viya4-iac-azure \
        destroy \
        -var-file /workspace/gel02-vars.tfvars \
        -state /workspace/gel02-vars.tfstate \
        -lock=false \
        -force \
        -input=false
    ```

* You can also do it fwth the azure CLI (more efficient)

    ```sh
    docker container  run  --rm \
        -v $HOME/jumphost/workspace/.azure/:/root/.azure/ \
        -v $HOME/jumphost/workspace/.kubeconfig_aks:/root/.kube/config \
        --entrypoint az \
        viya4-iac-azure \
        group delete --name ${STUDENT}-iac-viya4aks-rg --yes
    ```
